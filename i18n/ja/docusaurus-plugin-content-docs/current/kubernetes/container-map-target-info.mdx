### クバネティス状態の要約分析{#kube-summary-analysis}

クバネティス環境における各コンテナ、Pod、ノードのパフォーマンスとリソースに関する要約分析レポートを提供します。 クラスターとネームスペースのコンテナ、Pod、ノードのリソース割り当てと使用状況をリアルタイムでモニタリングし、レポート内の提案を参考にして効果的に対応できます。 

:::note 

現在、要約分析機能は韓国語でのみ提供しています。 

:::

#### コンテナ要約分析{#container-summary-analysis}

- 基本情報: コンテナの起動時間、再起動回数、Podおよびネームスペース情報を提供します。

- コンテナイメージ: 使用中のコンテナイメージと同じイメージを使用している他のコンテナの数およびリソース全体の使用量を確認できます。

- ノード情報: コンテナが実行されるノードのOS、IPアドレス、CPUコア数、メモリサイズなどのハードウェア情報を提供します。

- リソース使用状況: コンテナのCPUとメモリリソースの使用量、設定された限度(Limit)および要請(Request)の数と比較した使用率を提供します。
