2024年08月29日

- <Status priority='1'>Feature</Status> タグカウント`kube_node`のカテゴリにノード(Node)**capacity**と**allocatable**の関連指標を追加

  - `allocatable_ephemeral_storage`：ノードで使用できる一時ストレージ(ephemeral storage)を示します。

  - `capacityEphemeralStorage`：ノードの一時ストレージ(ephemeral storage)の総容量です。

  - `capacityPods`：ノードにスケジューリングできる最大Pod数です。

- <Status priority='0'>Fixed</Status>より正確なCPU割り当てデータを提供するため、`allocatable_cpu`指標でミリコア単位の値が漏れる現象を修正

:::
