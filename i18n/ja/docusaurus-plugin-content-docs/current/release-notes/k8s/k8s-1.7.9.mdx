2024年08月19日

- <Status priority='1'>Feature</Status> cgroup v2環境でrss関連メトリック指標の追加収集

  `container::mem_totalrss`：cgroup v2環境におけるコンテナのメモリRSS(Resident Set Size)を収集する指標です。 この指標はコンテナが実際に使用している物理メモリの量を示します。

- <Status priority='1'>Feature</Status>コンテナのrequest、limitの設定有無に対する指標追加

  - **Memory関連指標**

    - `container::memory_limit_contained`：コンテナのメモリlimitの設定有無を示す指標です。 値が`true`ならlimitが設定されており、`false`なら設定されていないことを指します。

    - `container::memory_request_contained`：コンテナのメモリrequestの設定有無を示す指標です。 値が`true`ならrequestが設定されており、`false`なら設定されていないことを指します。

  - **CPU関連指標**

    - `container::cpu_limit_contained`：コンテナのCPU limitの設定有無を示す指標です。 値が`true`ならlimitが設定されており、`false`なら設定されていないことを指します。

    - `container::cpu_request_contained`：コンテナのCPU requestの設定有無を示す指標です。 値が`true`ならrequestが設定されており、`false`なら設定されていないことを指します。

  - **Ephemeral-Storage関連指標**

    - `container::ephemeral_storage_limit_contained`：コンテナのephemeral-storage limitの設定有無を示す指標です。 値が`true`ならlimitが設定されており、`false`なら設定されていないことを指します。

    - `container::ephemeral_storage_request_contained`：コンテナのephemeral-storage requestの設定有無を示す指標です。 値が`true`ならrequestが設定されており、`false`なら設定されていないことを指します。

- <Status priority='1'>Feature</Status>ノードラベル指標をリストの形で追加

  - `kube_node::labelKey`：Kubernetesノードのラベルキー(Key)を収集する指標です。 ノードに適用されている各ラベルのキーを示します。

  - `kube_node::labelValue`：Kubernetesノードのラベル値(Value)を収集する指標です。 ノードに適用される各ラベルの値を示します。
