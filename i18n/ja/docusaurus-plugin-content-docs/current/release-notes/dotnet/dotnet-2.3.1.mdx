2024年08月26日

- <Status priority='1'>New</Status>トランザクション外部で発生するトレース情報の収集機能を追加

  - **expand_transaction_level** <span class='type'>Boolean</span>

    既定値`false`

    オプション値を`true`に設定すると、トランザクションの外部から発生するSQL、HTTPの呼び出しをトランザクションとして扱い、ヒットマップに表示します。

  - **expand_transaction_basetime** <span class='type'>Int</span>

    既定値`1000`(1秒)/最小値`100`(0.1秒)

    `expand_transaction_level`オプションが`true`である場合、設定した時間より応答時間が長くかかる場合にのみデータを収集します。 毎秒最大100件まで収集します。

- <Status priority='1'>New</Status> DB接続情報の表示：データベースを使用するトランザクションは、データベース接続情報を表示する機能を追加

- <Status priority='1'>Changed</Status>エラーでない場合でもトランザクションのステータスコードを表示するように修正
