2024年08月23日

## 共通

<Status priority='1'>Changed</Status> MySQL、PostgreSQL、Altibase、Oracle商品のエージェントオプション名を修正

- `xviewSkipSession` → `xview_min_sec`

- `tablespaceLock` → `tablespace_lock`

## Altibase

<Status priority='1'>New</Status>Altibase<code className="my">V2</code>機能を追加

- タグカウントパックに変更および収集情報を追加

- 下位バージョン互換が可能

## Oracle

<Status priority='0'>Fixed</Status>AIX環境で`v$osstat`メトリクスを収集しないように修正した内容の復元および収集されるメトリクスのみを処理するように修正

## PostgreSQL

<Status priority='0'>Fixed</Status>`pg_basebackup`修行中master処理されるバグを修正
