2024年09月11日

- <Status priority='1'>New</Status>エージェント設定ファイル(*whatap.ini*)より環境変数を優先に処理するオプションを追加

  - 既存のエージェント設定は、*whatap.ini*、**環境変数**の順に適用されます。

  - `whatap.use_env_first`(既定値`false`)オプションを設定すると*whatap.ini*ファイルの設定よりも環境変数を優先します。

    ```ini title="whatap.ini"
    whatap.use_env_first=true
    ```

  - `WHATAP_USE_ENV_FIRST=true`形式で環境変数を利用して設定できます。

- <Status priority='0'>Fixed</Status>`OKIND`環境変数の認識エラーを修正

  `OKIND`または`WHATAP_OKIND`環境変数として`okind`を設定できます。
