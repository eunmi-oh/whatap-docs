## 連携プロジェクトを削除する{#removelinkedproject}

1. <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_correlatedProjectManagement" className="uitext" />メニューに移動します。

2. <Cmdname sid="TTL07016" className="uitext" />セクションで連携を解除しようとするプロジェクトを削除するには、一覧の一番右にある![削除アイコン](/img/ico-trash.svg)ボタンを選択します。

3. 確認メッセージが表示されたら<Cmdname sid="delete" className="uitext" />ボタンを選択します。

<Cmdname sid="TTL07016" className="uitext" />セクションで削除したプロジェクトは除外されます。
