## エージェントダウンロード{#download-agent}

**アクセスキー**が発行されると、<Cmdname sid="JAVA-agent-setup-1-subject" className="uitext" />セクションで<Cmdname sid="download" className="uitext" />ボタンがアクティブになります。

1. <Cmdname sid="download" className="uitext" />ボタンを選択してから、WhaTapエージェントファイルをダウンロードしてください。

   :::tip

   次のコマンドを使用して、サーバーから直接にダウンロードすることもできます。

   ```bash
   wget https://api.whatap.io/agent/whatap.agent.java.tar.gz
   ```

   :::

2. モニタリング対象サーバーに接続してください。

3. 目的のモニタリング対象サーバーのパスにエージェントファイルを圧縮して解凍してください。圧縮を解凍したパスは`$WHATAP_HOME`になります。

   ```bash
   tar -zxvf whatap.agent.java.tar.gz
   ```
