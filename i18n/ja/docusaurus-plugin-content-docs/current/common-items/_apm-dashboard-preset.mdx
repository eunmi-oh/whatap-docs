## プリセットを設定する{#preset}

<InDoc product="java,php,python,nodejs,dotnet,golang,redis,mongodb,oracle-pro">

<ImgLang img="apm-dashboard-preset-list.png" desc="Preset" />

ダッシュボードからユーザーが設定したウィジェットの設定とレイアウト状態を保存して読み込むことができます。 ウィジェットのサイズを調整し、希望する場所に配置して新しいプリセットを作成できます。

:::note

- <Cmdname sid="default" className="uitext" />に設定されたプリセットは変更できません。

- <Cmdname sid="edit_project" className="uitext" />権限のあるメンバーのみがプリセットを保存または変更できます。 メンバー権限の詳細については、[次の文書](../project/project-structure#member-auth)を参照してください。

:::

</InDoc>

### 新しいプリセットを作成する{#newpreset}

1. ダッシュボードから、希望する形にウィジェットを配置してみてください。 サイズを変更して、頻繁に確認するウィジェットのみを配置することもできます。

2. 画面の右上に![保存アイコン](/img/ic-save.svg)ボタンを選択してください。

3. 新しいプリセット名を入力してください。

   <InDoc product="java,python,php,nodejs,dotnet,golang"><ImgLang img='apm-dashboard-preset-save.png' desc='プリセット保存' /></InDoc>

   <InDoc product="postgresql,mysql,oracle,oracle-pro,redis,mongodb,altibase,cubrid,mssql"><ImgLang img='db-dashboard-preset-save.png' desc='Preset' /></InDoc>

   エージェント選択履歴を一緒に保存するには、<Cmdname sid="ITM07345" className="uitext" />を選択してください。

4. <Cmdname sid="save" className="uitext" />ボタンを選択します。

プリセット一覧から新しく保存したプリセットを確認できます。

:::note

- 新しく作成したプリセットに変更が加えられた場合は、再度プリセットを保存してください。 ![保存アイコン](/img/ic-save.svg)ボタンを選択し、同じ名前でプリセットを保存します。 既存のプリセットに変更を上書きします。

- ダッシュボードの変更を保存せずに別のメニューに移動すると、変更は保存されません。

- プリセットはプロジェクト単位で保存され、他のユーザーと共有できます。

:::

### プリセットを削除する{#removepreset}

使用していないプリセットは、プリセット一覧から削除できます。 プリセット一覧から削除したい項目の右側に![削除アイコン](/img/ico-remove.svg)ボタンをクリックしてください。

:::note

<UI>Default</UI>プリセットは削除できません。

:::
