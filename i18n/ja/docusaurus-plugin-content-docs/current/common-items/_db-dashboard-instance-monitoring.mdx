## インスタンスモニタリング {#side_dbMonitoring}

- データベースサーバーの核心メトリクスとアクティブセッションに関する詳細情報をリアルタイムでモニタリングし、問題になる可能性のあるアラートを確認できます。関心のある重要メトリクスを選択して、全体的なシステム状況をモニタリングできます。

- リアルタイムモニタリングは、5秒間隔でパフォーマンス指標をモニタリングします。実行中の<Cmdname sid="active_sessions" className="uitext" />、<Cmdname sid="lock_tree" className="uitext" />、<Cmdname sid="process_info" className="uitext" />を確認できます。 

詳細については、[次の文書](instance-monitoring)を参照してください。
