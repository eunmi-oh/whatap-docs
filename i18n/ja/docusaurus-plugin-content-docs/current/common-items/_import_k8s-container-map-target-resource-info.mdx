- ***Deployment***: [Object Manifest](object-manifest) 情報を確認し、[Deployment リスト](deployment-list) メニューへ移動できます。

- ***ReplicaSet***: [Object Manifest](object-manifest) 情報を確認できます。 

- ***Pod***: [Object Manifest](object-manifest)、[Pod要約分析](container-map-target-info#pod-summary-analysis) 情報を確認し、[Podリスト](resource-pod-list) メニューへ移動できます。 

- ***ContainerId***: ***コンテナ情報***、[コンテナ要約分析](container-map-target-info#container-summary-analysis) 情報を確認し、[コンテナリスト](resource-container-list) メニューへ移動できます。

- ***Image***: ***コンテナイメージ***情報を確認し、[コンテナイメージ](container-images) メニューへ移動できます。 

- ***ServiceName***: [Object Manifes](object-manifest) 情報を確認し、[Serviceリスト](service-list) メニューへ移動できます。
