## agent_status_summary {#agent_status_summary}

エージェントの状態に関するメトリクスを10秒間隔で収集したカテゴリです。

### Fields {#agent_status_summary_filed}

<Xclude product="server,kubernetes">

| フィールド名        |    単位   | 説明                        | 備考               |
| ------------- | :-----: | ------------------------- | ---------------- |
| `inActTime`   | ミリ秒(ms) | エージェントが無効になったままの時間        | -                |
| `isActive`    |    -    | 現在のエージェントの活性化状態           | `true` / `false` |
| `isRestart`   |    -    | エージェントの再起動可否              | `true` / `false` |
| `lastActTime` | ミリ秒(ms) | 最後にエージェントが活性化された状態の時刻     | `0`: 無効化された場合    |
| `oid`         |    -    | プロジェクトに含まれる各エージェントの固有の識別子 | -                |
| `startTime`   | ミリ秒(ms) | エージェントが開始された時点でのタイムスタンプ   | -                |

</Xclude>

<InDoc product="server">

| フィールド名        |    単位   | 説明                        | 備考               |
| ------------- | :-----: | ------------------------- | ---------------- |
| `inActTime`   | ミリ秒(ms) | エージェントが無効になったままの時間        | -                |
| `isActive`    |    -    | 現在のエージェントの活性化状態           | `true` / `false` |
| `isRebooted`  |    -    | サーバーの再起動可否                | `true` / `false` |
| `lastActTime` | ミリ秒(ms) | 最後にエージェントが活性化された状態の時刻     | `0`: 無効化された場合    |
| `oid`         |    -    | プロジェクトに含まれる各エージェントの固有の識別子 | -                |
| `startTime`   | ミリ秒(ms) | エージェントが開始された時点でのタイムスタンプ   | -                |
| `status`      |    -    | 現在のエージェントの状態              | -                |
| `uptime`      |  秒(sec) | サーバーが稼働している時間             | -                |

:::note

`status`フィールドの値は、次のように区切ります。

- `ok`: サーバーが有効になっている正常な状態です。

- `inactive`: サーバーが無効になっている状態です。

- `warning`: サーバーに警告(warning) イベントが進行中の状態です。

- `fatal`: サーバーに危険(critical) イベントが進行中の状態です。

- `paused`: エージェントがデータの収集を一時停止した状態です。

:::

</InDoc>
