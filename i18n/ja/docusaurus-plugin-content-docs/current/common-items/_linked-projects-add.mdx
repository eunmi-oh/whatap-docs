## 連携プロジェクトを追加する {#addlinkproject}

1. <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_correlatedProjectManagement" className="uitext" />メニューに移動します。

2. <Cmdname sid="TTL07015" className="uitext" />セクションで<Cmdname sid="project_selection" className="uitext" />入力欄を選択します。

   <ImgLang img="linked-project-management-add-project.png" desc="相関プロジェクト管理" />

3. 選択できるプロジェクトの一覧が表示されたら、連携するプロジェクトを選択します。 文字列を入力して、一致するプロジェクトを検索もできます。

4. プロジェクトを1つ以上選択した後に有効になった<Cmdname sid="add" className="uitext" />ボタンを選択します。

<Cmdname sid="TTL07016" className="uitext" />に選択したプロジェクトが追加されます。

:::note 

連携プロジェクトを追加するためには、プロジェクト**修正**権限が必要です。 **修正**権限がある場合<Cmdname sid="side_correlatedProjectManagement" className="uitext" />メニューで<Cmdname sid="add" className="uitext" />ボタンが有効になります。 

:::
