## カラムを設定する{#setcolumn}

<Cmdname sid="TTL08445" className="uitext" />でテーブルヘッダーカラムを非表示にしたり、必要な項目を追加できます。 カラムの順序を変更することもできます。 ![カラムアイコン](/img/ic-column.svg) <Cmdname sid="BTN06441" className="uitext" />ボタンを選択します。

<ImgLang img="browser-statistics-setting-column.png" desc="カラム設定" />

:::note

左の一覧を右の順、または右の順にソートするには、![ソートアイコン](/img/ic-sort-up.svg) / ![ソートアイコン](/img/ic-sort-down.svg)ボタンを選択します。

:::

### カラムの追加

1. 左側の一覧で追加するカラム項目を選択します。

2. 選択したカラム項目が右側の一覧に追加されたことを確認します。

3. ご希望のカラム項目をすべて追加したら<Cmdname sid="confirm" className="uitext" />ボタンを選択します。

:::note

- すべてのカラム項目を選択するには、左の一覧の上段にある<Cmdname sid="select_all" className="uitext" />をクリックします。

- カラム項目の詳細については、[次の文書](#columninfo)を参照してください。

:::

### カラムを削除する

1. 右の一覧で削除するカラム項目の右側に![削除アイコン](/img/ico-remove.svg)ボタンを選択します。

2. 左側の一覧で削除したカラム項目のチェックボックスが解除されたことを確認します。

3. ご希望のカラム項目をすべて削除したら<Cmdname sid="confirm" className="uitext" />ボタンを選択します。

### カラム順序の変更

次に提供する機能により、カラムの順序を変更したり、必要な項目を固定できます。

- 右側の一覧で順番を変更する項目をドラッグして、目的の位置に移動してカラムの順番を変更できます。 

- カラム項目の右側の![ピンアイコン](/img/ic-pin.svg)ボタンを選択します。 選択したカラム項目は最上段に固定されます。 固定されたカラム項目は、テーブル一覧の一番左に固定されます。

すべての設定を完了した後は<Cmdname sid="confirm" className="uitext" />ボタンを選択します。

### カラム設定を初期化する

変更をすべてキャンセルして初期化するには、![初期化アイコン](/img/ico-reset.svg) <Cmdname sid="initialized" className="uitext" />ボタンを選択します。 初期化設定を反映するには、<Cmdname sid="confirm" className="uitext" />ボタンを選択します。
