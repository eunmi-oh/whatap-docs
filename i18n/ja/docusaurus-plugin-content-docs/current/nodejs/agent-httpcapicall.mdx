Node.jsエージェントのHTTPC、API Callのオプションを提供します。

- **profile_error_httpc_time_max** <span class='type'>Int</span> <ScrollToLink anchor='profile_error_httpc_time_max' />

  既定値`10000`

  HTTPCの実行時間が指定した値を超えると、**SLOW_HTTPC**エラーで処理します。`0`に設定するとエラー処理されません。

- **hook_httpc_patterns** <span class='type'>String</span>

  HTTP Callを実行するクラスを設定します。

- **httpc_not_found_ignore** <span class='type'>Boolean</span>

  既定値`false`

  Not Foundエラーが特定の時間(`httpc_not_found_ignore_time`)に50件以上発生すると、モニタリング検出を無視します。

- **httpc_not_found_ignore_time** <span class='type'>Milisecond</span>

  既定値`300000`

  `httpc_not_found_ignore`値が`true`である場合、Not Foundエラーを無視する時間を設定します。
