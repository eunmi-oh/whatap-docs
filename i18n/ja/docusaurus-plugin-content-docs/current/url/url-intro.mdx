WhaTap URL Monitoringは、エンドユーザーがウェブサイトのアクセスできるかどうかを判断するためのサービスを提供します。 ユーザーの視点からウェブサイトへのアクセスの正常なブラウザへのアクセスを通じて判断するためのサービスを提供します。WhaTapのモニタリングエージェントがサイトへのアクセスを行い、接続の有無を提供します。 サイトへのアクセスは、WhaTapのモニタリングエージェントがインストールされているサーバーから接続を試みます。 接続テストの結果は、http呼び出しレスポンスコードとレスポンス時間を提供します。 

ユーザーは、WhaTapのURLモニタリングサービスを個数制限なしにすべての機能を無料で使用できます。 モニタリングが必要なウェブサイトを登録し、ウェブサイトの点検を開始してください。 

## 主な機能

- **グローバルサービス向けのURLチェック**

  北米、ヨーロッパ、アジア、韓国等のサービス拠点を選択してモニタリングを行うことができます。

- **フリーサービス**

  WhaTapのURLモニタリングサービスのすべての機能を無制限に無料で提供します。

- **サイトアベイラビリティチェック**

  エンドユーザーのサイト接続の異常有無をHTTPステータスコードを通じてリアルタイムで確認します。

- **サイト性能モニタリング**

  北米、ヨーロッパ、アジア、韓国等に位置するサーバーから、サイトにアクセスするまでにかかる時間を確認します。

- **URL毎のアラート設定**

  Email、モバイルプッシュ、サードパーティープラグインを使用したアラート通知が設定できます。

  :::note

  URLモニタリングサービスは、SMS通知を提供していません。

  :::

- **WhaTap サービス統合ダッシュボードの構成**

  サーバー、アプリケーション、データベースなど、WhaTapの他のサービスとともにダッシュボードを構成できます。
