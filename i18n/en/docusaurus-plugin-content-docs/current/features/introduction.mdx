WhaTap newly introduced **Feature Project** (<code class='Features'>Features</code>) so that users can easily check whether the desired software supports monitoring and easily start monitoring. The Feature Project provides an optimized monitoring solution by repackaging existing monitoring products to fit the software to monitor. Manage your services for stability and performance more effectively through the WhaTap's Feature Project. 

- **Integrated monitoring experience**

  **Feature Project** not only provides easy installation through a dedicated menu, but also allows you to easily integrate and manage the monitoring configuration. Experience the integrated monitoring with user-friendly interfaces. 

- **Intuitive user experience**

  **Feature Project** not only allows you to easily check whether your software is compatible, but also provides a customized dashboard for the software. Monitor your software easily without cumbersome settings.

## KAFKA

KAFKA is a distributed streaming platform used for data stream processing. WhaTap provides the **_KAFKA_** monitoring with the **Feature Project** method. It traces the overall performance and health of the KAFKA cluster to help you find and resolve issues early.

## Aerospike

Aerospike is a high-performance NoSQL database optimized for real-time data processing and large-scale data analysis. WhaTap provides the **_Aerospike_** monitoring through **Feature Project**. It traces the cluster performance, response time, data consistency, and more in real time to help maintain and optimize the system stability. 

## Apache Pulsar

Apache Pulsar is a scalable, high-performance, open source messaging and streaming platform. WhaTap provides the **_Apache Pulsar_** monitoring with a **Feature Project** method. It helps enable efficient tracing of clusters.
