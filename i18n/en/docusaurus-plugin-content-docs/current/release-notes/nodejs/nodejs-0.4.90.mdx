February 5, 2024

- <Status>Feature</Status> Added the agent setting options.

  - **httpc_not_found_ignore** <span class='type'>Boolean</span>

    Default `false`

    If more than 50 Not Found errors occur in a certain time (`httpc_not_found_ignore_time`), monitoring detection is ignored.

  - **httpc_not_found_ignore_time** <span class='type'>Milisecond</span>

    Default `300000`

    If the value of `httpc_not_found_ignore` is `true`, set the time to ignore the Not Found errors.

- <Status>Fixed</Status> Fixed the issue where the Mongoose tracking does not work.
