- <Status>New</Status> Added the agent configuration options to collect transaction statistics by IP address.

  > **stat_ipurl_enabled** <span class='type'>Boolean</span>, Default `false`

- <Status>Changed</Status> Modified to apply the license and server host to both environment variables and *whatap.conf* file.

  When applied to both the environment variables and *whatap.conf* file, they are configured based on the environment variable's value.

- <Status>Changed</Status> Modified the ONODE value creation logic when setting the ONODE NAME.
