## Common

<Status priority='1'>Changed</Status> Changed the real-time call API (`DBX_USER_SCRIPT`) name to `DBX_SCRIPT_RUN`.

## PostgreSQL

<Status priority='1'>Changed</Status> Changed the pg table size collection function from `pg_relation_size()` to `pg_table_size()`.
