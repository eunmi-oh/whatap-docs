November 22, 2024

- <Status priority='1'>New</Status> Added the feature to enable and disable data collection for etcd monitoring.

  - Collecting whether the etcd reader exists

  - Collecting the etcd reader change trend

  - Collecting the etcd commit & apply trend

- <Status priority='1'>New</Status> Added the feature to enable and disable the Foreign Project feature.

  - Collecting data connected with the monitoring enabled in external Kubernetes projects
