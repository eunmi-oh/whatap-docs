November 25, 2024

- <Status priority='1'>New</Status> Added the agent options.

  **webservice_method_prefix_realtime** <span class='type'>Boolean</span>

  Default `false`

  If you set the value to `true`, the target of `webservice_method_prefix` is processed in real time as the previous.

- <Status priority='1'>Changed</Status> Changed the processing method for the transactions collected through the `webservice_method_prefix` option.

  - Modified to ignore the `webservice_method_prefix` methods collected within transactions to prevent errors caused by recursive calls.

  - Modified to ignore unfinished transactions.

- <Status priority='1'>Changed</Status> Merged installers for .NET Framework and .NET Core environments provided for separate downloads.
