<Xclude product="oracle-pro">

:::note

- The server to install the agent on must have Java 8 or later installed.

- The database server must allow inbound connections so that the DBX agent can access the DB port.

- It is required to open the outbound on the port 6600 so that the DBX agent can send data to the WhaTap server.

:::

</Xclude>
