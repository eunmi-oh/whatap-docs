- If you select ![number 1](/img/number-01.png) <Cmdname sid="TTL06202" className="uitext" />, ![number 2](/img/number-02.png) <Cmdname sid="TTL06203" className="uitext" />, and <Cmdname sid="instance" className="uitext" />, the data is automatically applied to all widgets based on the selected items.

- To see the meaning of the metric displayed in the widget, select ![Information icon](/img/ic-info-circle.svg) next to the name or see [the following](metrics-data-list).

  :::note

  The tooltip of the ![Information Icon](/img/ic-info-circle.svg) button is not supported in multiple languages.

  :::

- To change the position with another widget, select and drag the upper part of the widget. However, the size of the widget cannot be changed.

- The left of the ![number 3](/img/number-03.png) <Cmdname sid="active_sessions" className="uitext" /> table displays the data for the <Cmdname sid="TTL06202" className="uitext" />, and the right of the table displays the data for the <Cmdname sid="TTL06203" className="uitext" />.

- In the ![number 3](/img/number-03.png) <Cmdname sid="active_sessions" className="uitext" /> table, the text colors are changed black → <span class='slow'>orange</span> → <span class='vslow'>Red</span>, which means that the performance of the session is getting slower.
