## Monitoring a Database Instance {#side_dbMonitoring}

- You can monitor key metrics and the details on active sessions of the database server in real time and check the warnings that may be problematic. You can select desired key metrics and monitor the overall system status.

- The real-time monitoring checks the performance metrics at an interval of 5 seconds. You can check the <Cmdname sid="active_sessions" className="uitext" />, <Cmdname sid="lock_tree" className="uitext" />, and <Cmdname sid="process_info" className="uitext" />. 

For more information, see [the following](instance-monitoring).
