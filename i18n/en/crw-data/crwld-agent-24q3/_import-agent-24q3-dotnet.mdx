#### <code class='changelog-service'><a href="https://docs.whatap.io/release-notes/dotnet/dotnet-2_2_9">.NET Agent v2.2.9</a></code> <code class='changelog-date'>July 5, 2024</code>

- <code class="Feature">Feature</code> Supporting multi-transaction monitoring: You can trace calls between the application services registered as projects on the WhaTap monitoring platform.

#### <code class='changelog-service'><a href="https://docs.whatap.io/release-notes/dotnet/dotnet-2_3_1">.NET Agent v2.3.1</a></code> <code class='changelog-date'>August 26, 2024</code>

- <code class="New" type="1">New</code> Added the feature to collect traces that occur outside of transactions.

- <code class="New" type="1">New</code> Display DB connection information: Added the feature to display database connection information for transactions using the database.

- <code class="New" type="1">New</code> Added the agent option to set suffix of URLs to exclude from transaction collection.
